Installation

Traverse is a Rust workspace. Clone and build with Cargo.

install
$ git clone https://github.com/traverse-framework/traverse
$ cd traverse && cargo build

Finished. All crates compiled.

$ cargo test
test result: ok. 100% coverage · 0 failed

Requirements

Rust 1.94 or higher
Node.js v20+ (for React demo only)
wasm32-wasi target (rustup target add wasm32-wasi)
