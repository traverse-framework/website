Where to start
Building for AI agents?
You want Claude, Cursor, or another MCP-compatible client to call your capabilities.
→
MCP Server Setup Get the stdio server running and test with tools/list in under 15 minutes.
Adding to an existing React app?
You have a frontend and want to call governed capabilities from it.
→
React Integration No Rust required. The browser adapter handles everything behind a simple HTTP endpoint.
First time here?
You want to understand what Traverse does before writing any code.
→
Quickstart Run the expedition demo in under 5 minutes. Then come back and pick a guide.
Writing a new capability from scratch?
You have a business function you want to govern and deploy as WASM.
→
Quickstart first Then Author a Capability when it is published. Track it on GitHub.
